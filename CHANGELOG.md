# CHANGELOG

ReserveLine Pro の主な変更履歴です。

## v0.2.2：公開リポジトリ整備

- `CHANGELOG.md` を追加
- 動作確認シナリオを追加
- READMEから関連ドキュメントへ移動できるリンクを追加
- `.gitignore` の公開除外設定を再確認
- テスト・構文チェック・起動確認を再実行

## v0.2.1：管理画面改善

- 管理画面の日本語UI文言を改善
- 本日の予約、今週の予約、キャンセル、リマインド予定の件数カードを追加
- 画面確認用の予約データ作成機能を追加
- 製品概要ドキュメントを改善
- READMEの説明とよくあるエラーを改善
- Streamlit管理画面の実起動を確認

## v0.2：実用MVP改善

- 同じ日時への重複予約を防止
- 過去日時、営業時間外、定休日の予約を拒否
- 営業時間、営業日、タイムゾーンを `.env` で変更可能に変更
- Streamlit管理画面に簡易ログイン・ログアウトを追加
- CSV出力の列名を日本語化
- 日付入りのCSVファイル名に変更
- 予約バリデーションのテストを追加

## v0.1：初期版

- LINE Bot風の予約受付
- FastAPI Webhook
- Streamlit管理画面
- SQLiteへの予約データ保存
- LINE APIなしで確認できるデモモード
- 予約作成、確認、キャンセルの基本テスト
