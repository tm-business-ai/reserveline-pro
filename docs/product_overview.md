# ReserveLine Pro 製品概要

## 概要

ReserveLine Proは、小規模事業者向けのLINE予約受付・予約管理システムです。LINEでの予約受付、予約確認、キャンセル受付を想定し、店舗側は管理画面から予約状況や予約データを確認できます。

LINE公式アカウント接続前でも動作確認可能な構成にしており、予約受付の流れ、管理画面、予約データのダウンロード、予約前のお知らせ予定を確認できます。

## 解決できる課題

- LINE、電話、店頭などに分かれた予約受付を整理したい
- 予約内容の転記作業を減らしたい
- 同じ日時への重複予約を防ぎたい
- 予約確認やキャンセル対応を分かりやすくしたい
- 予約前のお知らせ対象を一覧で確認したい
- 予約一覧をCSVで保存し、店舗業務に活用したい

## 主な機能

- LINE予約受付を想定したWebhook API
- メニュー一覧の表示
- 直近予約の確認
- 直近予約のキャンセル
- 電話・店頭予約の登録
- 予約状態の変更
- 本日の予約・今週の予約表示
- キャンセルされた予約の表示
- 予約データのCSVダウンロード
- 予約メニューの設定
- 予約前のお知らせ予定表示
- LINE公式アカウント接続前の動作確認
