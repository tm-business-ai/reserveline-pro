# ReserveLine Pro 操作マニュアル

## 1. 事前設定

`.env.example` をコピーして `.env` を作成します。

```powershell
copy .env.example .env
```

管理画面を使う前に、`.env` の `ADMIN_PASSWORD` を変更してください。

```env
ADMIN_PASSWORD=任意の管理者パスワード
```

営業時間と営業日は次で変更できます。

```env
BUSINESS_OPEN_TIME=09:00
BUSINESS_CLOSE_TIME=18:00
BUSINESS_DAYS=mon,tue,wed,thu,fri,sat
BUSINESS_TIMEZONE=Asia/Tokyo
```

## 2. 管理画面の開き方

PowerShellで次を実行します。

```powershell
streamlit run ui/admin_app.py
```

または、プロジェクトフォルダ内の `run_admin.bat` を実行します。

## 3. ログイン

管理画面を開くと「管理画面ログイン」が表示されます。

`.env` の `ADMIN_PASSWORD` に設定したパスワードを入力し、「ログイン」を押してください。

作業を終える場合は、サイドバーの「ログアウト」を押します。

## 4. 本日の予約

サイドバーで「本日の予約」を選ぶと、今日の予約を一覧で確認できます。

## 5. 今週の予約

サイドバーで「今週の予約」を選ぶと、今日から7日間の予約を確認できます。

## 6. 電話・店頭予約の登録

サイドバーで「電話・店頭予約の登録」を選ぶと、電話や店頭で受けた予約を、店舗側で登録できます。

入力する内容：

- 顧客名
- LINEユーザーID
- メニュー
- 予約日
- 予約時間
- 備考

予約登録時には、次のチェックが行われます。

- 過去日時は予約不可
- 営業時間外は予約不可
- 定休日は予約不可
- 同じ日時に有効な予約がある場合は予約不可

## 7. 予約状態の変更

予約IDを選び、次のいずれかに変更できます。

- 予約中
- 確定
- 仮予約
- 完了
- キャンセル
- 無断キャンセル

キャンセル済みの予約は、重複予約チェックの対象外です。

## 8. キャンセルされた予約

サイドバーで「キャンセルされた予約」を選ぶと、キャンセル扱いになった予約を確認できます。

## 9. 予約データをダウンロード

サイドバーで「予約データをダウンロード」を選ぶと、予約一覧を保存できます。

Excelで開けるCSV形式で保存できます。列名は日本語で出力され、ファイル名は `reservations_YYYYMMDD.csv` 形式です。

## 10. 予約メニューの設定

「予約メニューの設定」は、お客様が選べる予約メニューを登録・編集する画面です。

メニュー名、所要時間、料金、表示順、受付中かどうかを管理できます。

設定できる内容：

- メニュー名
- 所要時間（分）
- 料金（円）
- 表示順
- 受付中かどうか

メニュー名が空の場合は保存できません。

## 11. LINE予約の動作確認

サイドバーで「LINE予約の動作確認」を選ぶと、LINEでメッセージを送った場合の返信内容を管理画面から確認できます。実際の利用時は、お客様がLINEのトーク画面から予約します。LINE公式アカウントとつなぐ前でも、予約受付の動きを確認できます。

例：

```text
予約
予約 30分相談 2026-05-10 10:00
確認
キャンセル
```

## 12. 予約前のお知らせ予定

「予約前のお知らせ予定」は、予約したお客様へ、予約前に送るお知らせの予定を確認する画面です。

誰に、いつ、お知らせを送る予定かを一覧で確認できます。

LINE公式アカウントとつなぐ前でも、予約前のお知らせ対象の確認と処理の流れを確認できます。

## 13. 予約ルールの設定

サイドバーで「予約ルールの設定」を選ぶと、予約できる曜日、営業時間、予約間隔、受付締切、予約可能期間を変更できます。

設定できる内容:
- 営業曜日
- 営業開始時刻
- 営業終了時刻
- 予約間隔（分）
- 予約受付の締切時間（分）
- 予約可能期間（日）
- タイムゾーン

入力後に「予約ルールを保存する」を押すと、設定が保存されます。保存したルールは、LINE予約を想定した受付、電話・店頭予約の登録、空き時間の確認に反映されます。

## 14. 臨時休業日の設定

「予約ルールの設定」画面内の「臨時休業日の設定」から、通常の定休日とは別に予約を受け付けない日を登録できます。

入力する内容:
- 休業日
- 理由
- 有効にするかどうか

登録済みの臨時休業日は一覧で確認できます。有効／無効を変更したい場合は、対象の休業日IDを選び、状態を変更してください。有効な臨時休業日は予約可能枠から除外されます。

## 15. 空き時間の確認

サイドバーで「空き時間の確認」を選ぶと、指定した日付とメニューに対して予約可能な時間を確認できます。

確認する内容:
- 日付
- メニュー

予約可能な時間がある場合は、時間一覧が表示されます。予約できる時間がない場合は、「この日付で予約できる時間はありません。」と表示されます。

空き時間の判定では、営業曜日、営業時間、予約間隔、受付締切、予約可能期間、臨時休業日、メニュー所要時間、既存予約との時間帯重複を確認します。
